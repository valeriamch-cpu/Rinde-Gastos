# Rinde-Gastos

Primera versión web para cargar rendiciones de trabajadores y obtener un resumen inicial de:

- Tipo de gasto.
- Valor.
- Respaldos adjuntos (listado por nombre de archivo).

## Uso rápido

1. Abre `index.html` en tu navegador.
2. Completa trabajador, tipo de gasto, valor y adjunta respaldos.
3. Presiona **Agregar rendición** para ver el resumen y el total acumulado.
